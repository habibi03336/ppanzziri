# 6. 계산 기준 요약

| 지표 | 기준 | 범위 |
| --- | --- | --- |
| 현재 잔액 | 거래 | 전체 |
| 누적 수입/지출 | 거래 | 전체 |
| 잔액 그래프 | 거래 | 전체 / 최근 30일 |
| 하루 평균 비용 | 유효 | 전체 / 최근 30일 |
| 태그 비중 | 유효 | 최근 30일 |

# 7. 구조적 원칙

- 잔액은 항상 거래 기준이다.
- 하루 평균 비용은 유효 기준이다.
- 태그 비중은 유효기준, 최근 한 달 기준으로 해석한다.
- 최근성은 생존 감각을 드러내는 핵심 요소다.
